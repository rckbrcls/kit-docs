Polterbase is an optimized interactive CLI for managing Supabase CLI workflows with more speed, consistency, and discoverability. It is a productivity layer on top of the official `supabase` CLI.

Instead of memorizing command trees, you browse one unified board, add extra args interactively, attach global flags, and pin common workflows for one-click reuse.

## Installation

### Run one-off without installing

```bash
npx @polterware/polterbase@latest
```

### Install in a repository

```bash
npm install -D @polterware/polterbase
```

Then run it from that repository:

```bash
npx polterbase
```

### Install globally

```bash
npm install -g @polterware/polterbase
```

Then run:

```bash
polterbase
```

Use a repository install when you want the CLI version pinned to one project.
Use a global install when you want the same CLI version across every repository.

### Update

If installed globally:

```bash
npm install -g @polterware/polterbase@latest
```

If installed in a repository:

```bash
npm install -D @polterware/polterbase@latest
```

You can also update from inside Polterbase:

1. Go to the **Actions** section
2. Choose **Update Polterbase**
3. Choose **Current repository** or **Global install**
4. Confirm the npm update command

## Requirements

- **Node.js** >= 18
- **Supabase CLI** installed globally or locally in the current repository

Check your environment:

```bash
node -v
supabase --version
```

Install Supabase CLI: [Supabase CLI Guide](https://supabase.com/docs/guides/cli)

## Typical Flow

1. Choose a command from the unified board
2. Optionally pin/unpin with `→`
3. Choose suggested args or type custom extra args
4. Pick optional global flags
5. Confirm and execute
