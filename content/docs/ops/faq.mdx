## Common questions

<Steps>

### Do I need Supabase CLI?

Yes for repository and migration workflows. The Ops CLI wraps Supabase CLI commands for setup and migration operations.

### Can I use a packaged app without rebuilding?

Yes. Use the install script to download a release from GitHub and install it to `/Applications/`. The app presents a connection form on first launch.

### How do I switch to another Supabase project?

Relink the source checkout with `supabase link` and then run `pnpm ops db push` to apply migrations. For packaged apps, update the connection in app settings.

### What is the difference between env config and runtime config?

env config is the development fallback used in source checkouts (`.env.local`). Runtime config is the persisted desktop connection used by installed apps.

</Steps>
